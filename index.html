Welcome to GitHub Pages.

This tool simulates and Open Source Community with Intelligent Agents and Community Games. It is an application with a server-client model created with an educational purpose in the context of a Master's Thesis at Tampere University of Technology (TUT).

To run the application locally follow the next steps:

  • Download the source code.
  • Run the Server Applet
  • Run the Agents Applet and connect it.
  • Create a Random Community.
  • Run the Client Applet and connect it.
